EarthPy is a collection of IPython notebooks with examples of Earth Science related Python code. It can be tootorials, descriptions of the modules, small scripts, or just tricks, that you think might be usefull for others.

Contrubutions are very wellcome. To add your code to EarthPy, just send .ipynb file to koldunovn@gmail.com
